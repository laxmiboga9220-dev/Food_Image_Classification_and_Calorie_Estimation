# Food_Image_Classification_and_Calorie_Estimation
Food image classification and calorie estimation using EfficientNet.

#  Food Recognition and Calorie Estimation

##  Overview

This project uses deep learning to classify food images and estimate their calories. This model is based on EfficientNet and performs both Classification and Regraession task.

##  Dataset

* Food-101 dataset (10 classes selected)
* Images preprocessed and split into traingng and validation sets

##  Model

* EfficientNet-B0
* Outputs:

  * Food category (Classification task)
  * Calorie estimation (Regression task)

##  Technologies

* Python, PyTorch
* NumPy, Pandas
* Google colab

##  Metrics

* Accuracy (for classification)
* Mean Squared Error (for regression)

##  How to Run

1. Open in Google Colab
2. Load dataset (Drive or online)
3. Run all cells

##  Conclusion

The model effectively predicts food categories and provides reasonable calorie estimates using a multi-task learning approach.
